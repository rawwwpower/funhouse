# funhouse

The mothership for creative experiments. Iggy would be proud.

## Projects

### [ohara](./apps/ohara)
Personal knowledge library. Drop anything — links, screenshots, PDFs, notes — and AI organizes it for you. Named after the island of the Tree of Knowledge.

## Setup

```bash
pnpm install
pnpm dev:ohara
```

## Structure

```
apps/       → individual projects
packages/   → shared utilities (when needed)
```
